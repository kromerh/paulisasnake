# paulisasnake

This is a IBM Watson IoT powered snake terrarium monitor.

## Hardware

- NodeMCU Lua Lolin V3 Module ESP8266 ESP-12E
- DHT22 sensor
- jumper cables
- power supply
- breadboard(s)
